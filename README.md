# Frequency-Based Keyword Extraction

This project performs keyword extraction using TF and TF-IDF techniques on the Reuters-21578 dataset.

Steps:
1. Dataset parsing from SGML
2. Text preprocessing
3. Term Frequency extraction
4. TF-IDF keyword extraction
5. Export keywords to CSV
6. WordCloud visualization

Technologies:
Python, NLTK, Scikit-learn, BeautifulSoup
